# laravel-my-media-library

## 1. Installation
### create .env file for docker-compose.yaml
```bash
cp .env.example .env
```

### run init script
```bash
sh init.sh laravel-my-media-library
```

## 2. Open Admin Dashboard
http://localhost/admin/login
```
E-Mmail:  admin@example.com
Password: secret
```
